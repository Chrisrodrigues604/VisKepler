Instruções

Neste sistema, você pode explorar mapas em três diferentes escalas: Regiões (NUTS3), Municípios (NUTS4) e Freguesias (NUTS5). Utilize os botões à direita para selecionar a escala desejada. Se precisar de mais informações ou bases de dados, clique no botão "Dados" para fazer o download. Você também pode fazer upload dos seus próprios dados para compará-los com os dados disponíveis.

Lembre-se das funcionalidades essenciais do KeplerGL ao utilizar este sistema. Você pode personalizar a visualização do mapa, filtrar dados, explorar diferentes camadas e salvar seu trabalho em diferentes formatos.
